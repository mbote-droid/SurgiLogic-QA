# 🩺 SurgiLogic-QA
### Clinical Reasoning & Risk Stratification for General Surgery

SurgiLogic-QA is a Python-powered tool designed to assist **General Surgery Residents** in post-operative monitoring. It bridges the gap between clinical logic and automated data validation.

## 🚀 Technical Highlights
- **Decision Engine:** Maps clinical vitals to potential post-op complications based on Post-Op Day (POD).
- **Error Handling:** Robust validation for non-standard numerical inputs (C/F conversion logic).
- **Web Interface:** Fully functional UI built with **Streamlit**.

## 🛠️ Tech Stack
- **Python** (Logic & Data Processing)
- **IBM Data Science Principles** (Applied via IBM Professional Certification)
- **Streamlit** (Deployment)

## ⚠️ Medical Disclaimer
This is a **Proof of Concept (PoC)** for AI training and clinical reasoning purposes. It is NOT a clinical diagnostic tool.
